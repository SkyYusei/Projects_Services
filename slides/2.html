AWS Account Sign Up

Welcome to the projects of the Cloud Computing course! For the projects in this course, you will work on instances provisioned on Amazon EC2 (Elastic Compute Cloud) and utilize S3 (Simple Storage System) for persistent storage. In addition, we will be working with various other AWS (Amazon Web Services) technologies such as Elastic Block Store (EBS), CloudWatch, and DynamoDB, among others. For more information about AWS, please refer to the AWS website, or read the AWS Primer in this course. Let's get started by creating an AWS account.

AWS Account

The following video will outline the steps necessary to create your own AWS account and link it with the course account for payment purposes:

Video: AWS Sign Up

Please follow the steps outlined below in order to set up your AWS account for use with the course:

  1. Create a new AWS account at http://aws.amazon.com by clicking on Sign Up.
  2. Follow the onscreen instructions as outlined in the video. Please use your andrew email to sign up for AWS. You will need to enter a credit/debit card and address information as EC2 requires a credit card on file for use. A $1 (USD) hold may occur on the card during verification. However, your EC2 usage charges for the class will be paid by the Cloud Computing Course after you have completed the steps on this guide.
  3. Ensure that you sign up for only basic support when asked.
  4. Once you have completed the EC2 sign up, and completed the phone verification process, wait for EC2 authorization mail and activation. This can take up to one business day.
  5. You can now sign into your account at http://aws.amazon.com.
  6. Sign in to AWS and retrieve your AWS account number.
  7. Update your profile page, click on "Change Account" and enter your AWS account information and the email address you used to sign up for the account in the relevant field.
  8. In one or two business days, you should receive a consolidated billing request in your email, please click and approve the request you get by email from AWS so that your EC2 usage can be charged to the university.

Warning

You must provide your AWS account information and the email address used to create the account so that the course staff can issue an AWS account link request. Linking your AWS account to the course AWS account is mandatory otherwise you will not be able to gain access to course-specific resources on the cloud.

After we issue the AWS account link request, you must click and approve the consolidated billing request that you get by email. Failure to do so will result in your account not being linked and charges will be forwarded to the credit/debit card on file with AWS.

Grade Penalties on Excessive Use

You are solely responsible for the usage of your AWS account. Each project in the course has an associated budget which must be followed strictly. Usage reports will be monitored by the instructors and excessive use will be grounds for grade penalties.

Your AWS account that you create and link to the course should be used exclusively for the projects in this course. This account must not be used for any private work or for other courses/projects at CMU or elsewhere.

Dropping the Course

You will be removed from the AWS consolidated billing if you drop the course. You will receive an email indicating that you will be removed from the billing account. After you have been removed from the course account, any resources that you have left running will be billed on the card you have used to sign up for AWS. Please ensure that you terminate all resources and delete your account, if you do not plan to use AWS in the future.

AWS Educate Account

Students in this course have to sign up for an AWS Educate Account. Please follow the steps outlined below in order to set up your AWS Educate account for use with the course:

  1. Visit the Apply for AWS Educate page.
  2. Click on the Apply for AWS Educate for Students.
  3. Select Student and click on Next.
  4. Fill out the application with your information and fill out your AWS Account ID when asked, click on Next
  5. As you fill out your application, you should receive an email. Verify your email address on the next page and click on Next.
  6. Accept the terms and conditions on the next page and finish the application.
  7. You should receive an email confirmation from AWS indicating that you have successfully joined the AWS Educate program.

Azure Account Setup

In addition to AWS, we are offering students the ability to get familiar with Azure, the cloud service from Microsoft. For more information about Azure, please refer to the Azure Website, or read the Azure Primer in this course.

Azure Account Sign Up

Please follow the steps outlined below in order to set up your Azure account for use with the course:

Warning

Please make sure you DO NOT use your Carnegie Mellon issued andrew email id as your id when you sign up with Azure.

  1. Obtain the Azure sign-up pass using the form below. You should find your Submission Password as a clickable button at the top of this page.


  2. Create a new Azure account at http://www.microsoftazurepass.com. Enter the region as your current country location and use the Azure sign-up pass and click Activate.
  3. In the next screen, you will be required to use a Microsoft ID. You may use an existing Microsoft/Outlook/Hotmail email ID or sign up for one.
  4. Next, Azure may sometimes require phone verification, please enter a valid phone number to receive a confirmation code. Enter the confirmation code and agree to the Azure terms of service to continue.
  5. Now, Azure will set up your account, which may take a few minutes.
  6. You can now log into your azure account by visiting http://portal.azure.com.

Usage Responsibility

The Azure pass provided entitles you to $100 per month for 6 months. You will be responsible for excessive usage on your account. Monitor your usage and ensure that you terminate all resources on Azure after you have completed using them.
